# 🤖 로보레인져스 : 스마트 안내 로봇 프로젝트 (RODY)

> **대형 복합 쇼핑몰 내 길안내, 물품보관, 순찰까지 가능한 멀티태스킹 로봇 개발 프로젝트**  
> *(KG KAIROS 2기 통합 프로젝트 / 2024년 하반기)*

---

## 🔖 프로젝트 개요

| 항목 | 내용 |
| --- | --- |
| **프로젝트명** | 로보레인져스 : 스마트 안내 로봇 |
| **수행 기간** | 2024년 하반기 |
| **목적** | 쇼핑몰, 병원 등에서 안내/보관/순찰 서비스를 통합 제공하는 멀티 기능 로봇 구현 |
| **역할** | AGV 제어, 카메라 캘리브레이션, 웹 인터페이스 개발, 로봇 제어 통신, AI 기반 물체 인식 등 |
| **사용 기술** | AGV, PLC, ESP32, Python, OpenCV, Flask, WebSocket, HTML, YOLOv8 |

---

## 🧩 시스템 구성도

- 📷 **RGB 카메라** : AGV 영상 인식, ArUco 마커 기반 위치 보정 및 라인트레이싱  
- 🤖 **AGV** : 라인트레이싱 기반 경로 주행 + 마커 인식 기반 이동 전환  
- 🔐 **물품 보관함** : 로봇팔 + 카메라를 통한 물체 인식 및 위치 제어  
- 🌐 **웹 페이지** : Flask + HTML 기반 실시간 제어 UI, 영상 확인, 목적지 설정  
- 🔌 **ESP32 / PLC** : 웹서버 ↔ 로봇 ↔ AGV 간 명령 송수신  
- 🔁 **WebSocket 통신** : 비동기 제어 및 실시간 데이터 전송  
- 🛡 **순찰 모드** : 정해진 루트 이동 및 카메라 영상 송신, 이상 시 관리자 페이지에 저장  

## 📊 **시스템 흐름**

[Camera] → [AI 모델 (YOLOv8)] → [제어 서버 (Flask + WebSocket)] → [AGV / 로봇팔 / ESP32 / PLC]

## 🚀 주요 기능

### 1️⃣ AGV 길 안내
- 라인트레이싱 + ArUco 마커 기반 경로 이동  
- 목적지 선택 시 사용자가 손잡이를 잡으면 자동 주행  
- 직선/좌회전/우회전 판단 알고리즘 구현  

### 2️⃣ 물품 보관 및 찾기
- YOLOv8 + 로봇팔을 통한 실시간 물품 감지  
- 카메라 좌표 `(cx, cy)`를 로봇팔 좌표계 `(x, y, z)`로 변환  
- 비밀번호 기반 웹 UI로 보관 및 반환 관리  

### 3️⃣ 순찰 모드
- 외부인 감지 시 자동 촬영 → 관리자 페이지 실시간 전송  
- 이상 상황 발생 시 영상 저장 및 경고 표시

## 🛠 코드 구성

### 🧠 주요 Python 파일
| 파일명 | 설명 |
| --- | --- |
| `agv_control_gui.py` | AGV 경로 제어 GUI |

### 💻 HTML UI 페이지
- `page_main.html` – 메인 제어 화면

- `page_admin.html` – 관리자 페이지  
- `page_admin_login.html` – 관리자 로그인  

### 🧮 기타 리소스
